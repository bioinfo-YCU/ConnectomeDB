CDB2025 User Guide: Overview

This tutorial provides a step-by-step guide for exploring and utilizing the ConnectomeDB platform, designed for the exploration of manually reviewed high-quality ligand–receptor interactions across human and 13 additional model organisms. The database integrates our proprietary Ligand-Receptor pairs combined with annotations from Ensembl BioMart, UniProt, PubMed, and other public resources.

Key features demonstrated in this tutorial include:

Flexible multi-gene search: Query one or more genes to identify ligand–receptor pairs and their orthologs.

Cross-species ortholog mapping: Select your model organism to explore orthologous interactions across human and supported species.

Interactive data exploration: Sort, and customize table views directly in the browser.

Bulk versioned data downloads: Access complete, date-stamped datasets for each species, ensuring reproducibility and historical reference.

Evidence tracking: Link to supporting literature (PubMed) and functional annotations.

This tutorial is structured into self-contained modules so you can quickly locate the information you need—whether you are a new user looking for basic navigation help or an experienced researcher exploring advanced search and download features.

To learn more about the database, visit the ConnectomeDB documentation. Methods are described in detail in the Methods.
